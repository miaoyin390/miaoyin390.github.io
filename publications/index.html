Towards Efficient Tensor Decomposition-Based DNN Model Compression with Optimization Framework

We propose a systematic framework using the Alternating Direction Method of Multipliers (ADMM) for tensor decomposition-based model compression, specifically to solve the performance degradation problem in compressing CNNs. The framework treats model compression as an optimization problem with constraints on tensor ranks, gradually introducing low tensor rank characteristics to a standard deep neural network model during training. This model is then converted and fine-tuned to a tensor train format, yielding a high-accuracy, highly compressed model. Tests show substantial compression ratios and improved accuracy across different neural network models and tasks.

IEEE/CVF Conference on Computer Vision and Pattern Recognition, 2021
